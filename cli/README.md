## Installation

```bash
# Navigate to repository
cd esoterica/cli

# Make executable (Unix/Mac)
chmod +x esoterica.py

# Run directly
python esoterica.py

# Or create alias (add to .bashrc/.zshrc)
alias esoterica='python /path/to/esoterica/cli/esoterica.py'
```
