# /zeit — Read the Present

You are performing a zeitgeist reading. This is a consciousness technology: reading the surface of events to discover what's actually moving underneath. The correspondence filesystem (as above, so below) means every pattern indexes at every scale — personal, civilizational, cosmic.

## Architecture: gather in isolation, synthesise from distillate

Don't pull raw world-signal into the main context and then write — that pattern trips Anthropic's cyber-content classifier and kills the whole response mid-run (a week of conflict/breach/exploit headlines held in one context and narrated in one long generation reads as violative content, even though this is contemplative journalism). Instead, **gather agents** touch the raw feed in isolated contexts and return *neutral factual digests*; the **main turn synthesises only from those digests**. A blocked agent costs one channel, not the reading.

## Step 1: Deploy Gather Agents (Parallel)

Launch **two gather agents simultaneously** using the Agent tool (`subagent_type: general-purpose`). Each runs its own WebSearch/WebFetch and returns a neutral digest. Foreground, and wait.

Give both agents this **register contract** verbatim, then its channels:

> You are gathering raw world-signal for a contemplative news reading. Return a neutral, factual, journalistic digest — nothing else. Do NOT write any files.
> **Register rules (hard):** For any story touching security, cyber, conflict, weapons, surveillance, or crime, report ONLY what happened and why it matters. Never include operational or technical detail — no methods, tools, code, vulnerabilities, exploit steps, malware behaviour, or capability specifics. Headline-level only. You are summarising events, not explaining how anything was done. No enablement of any kind.
> **Format:** Under each channel heading, 3–5 bullets. Each bullet: `**[plain headline]** — [1–2 sentence factual summary: who/what/when/where]. (source: URL)`. Plain language, no amplification, no analysis — analysis happens later, not here.

**Agent A — material world:**
1. Geopolitical — "major world news today" (conflicts, diplomacy, power shifts, elections, treaties)
2. Economic — "global economy financial markets news today" (markets, policy, currencies, trade, systemic shifts)
3. Ecological — "climate environment ecology news this week" (earth systems, species, weather, energy transition)

**Agent B — frontier world:**
4. Scientific — "scientific breakthroughs discoveries research 2026" (papers, findings, paradigm shifts, space, physics, biology)
5. Technological — "AI technology breakthrough news this week" (releases, capabilities, regulation, infrastructure)
6. Cultural — "culture society trends news this week" (movements, art, discourse shifts, collective mood)

**Query hygiene:** never let a search string pair hacker/exploit/breach/malware/attack with how/tool/build/code. Important security stories are still reported — at headline level, per the contract.

## Step 2: Read the Pattern

With both digests in front of you — and only the digests, never raw feeds — pause. Don't summarize yet. Look for:

- **Convergences**: where do independent events rhyme? What theme appears across unrelated domains?
- **Tensions**: what polarities are active? What's pulling apart, and what's that tension *generating*?
- **Emergences**: what's appearing for the first time, or crossing a threshold? The genuinely new.
- **Absences**: what's conspicuously missing from the news? What's the dog that didn't bark?

If an agent came back blocked or empty, note the gap and proceed with what you have.

## Principles

- **Gather in isolation, synthesise from distillate.** The main turn never holds the raw feed. Agents distill; you read the distillate.
- **Register is the fix, not censorship.** Cover whatever is actually happening — at the level of significance and meaning, never operational detail.
- **Fidelity first**: report what's actually happening. Don't bend events to fit a narrative.
- **Mark your territory**: signal is factual, depth is analytical, state is interpretive. Keep these registers distinct so the reader can choose their depth.
- **Density over decoration**: every sentence earns its place. No filler, no throat-clearing.
- **The reading is not the news**: any aggregator can list events. The value is in the pattern recognition — what the *combination* reveals that no single event shows alone.
- **Surprise yourself**: if the reading confirms what you already expected, look harder. The zeit always contains something the map didn't predict.
